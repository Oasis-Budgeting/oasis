## 2024-04-04 - Global Navigation Accessibility
**Learning:** `aria-expanded` and `aria-keyshortcuts` state indicators are essential complements to `aria-label`s for icon-only global navigation and action buttons (like the FAB) to provide context for screen readers.
**Action:** Always add state indicators (`aria-expanded`, `aria-pressed`, `aria-keyshortcuts`) alongside `aria-label`s for icon-only buttons that toggle UI elements or have keyboard shortcuts.
