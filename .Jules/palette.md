## 2025-03-09 - Add ARIA Labels to Icon-Only Floating Action Buttons
**Learning:** Core layout controls, such as floating action buttons and mobile menu toggles, often rely entirely on icons to save space, but are frequently overlooked for accessibility. The primary floating action button in this app lacked an ARIA label, meaning screen readers would read it as a generic "button" instead of its intended "Quick Add Transaction" function. Similarly, the mobile menu toggle lacked state (`aria-expanded`) and an accessible label.
**Action:** Always ensure that icon-only buttons, especially those that act as primary actions or navigation toggles, have an explicit `aria-label`. For toggle controls, include `aria-expanded` to communicate the current state to assistive technologies.
